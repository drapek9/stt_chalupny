# Webová stránka STT s.r.o. (Chalupný a syn)

Moderní, responzivní webová stránka pro firmu STT s.r.o., která se specializuje na opravy a servis plynových spotřebičů.

## Struktura webu

- **index.html** - Úvodní stránka s velkým CTA a představením firmy
- **sluzby.html** - Přehled služeb (opravy, servis, instalace, revize)
- **pribeh.html** - Historie firmy a hodnoty
- **reference.html** - Ukázky realizovaných zakázek
- **kontakt.html** - Kontaktní údaje, adresa, mapa
- **tipy.html** - Tipy a rady pro bezpečné používání plynových spotřebičů
- **styles.css** - Společné CSS styly
- **script.js** - JavaScript pro navigaci a interaktivitu

## Design

- **Minimalistický a přehledný** - strukturované bloky, jasné CTA
- **Responzivní** - funguje na mobilu, tabletu i desktopu
- **Moderní a profesionální** - důvěryhodný vzhled
- **Barevně neutrální** - modrá primární barva s červeným akcentem pro CTA

3. Pro produkční nasazení nahrajte všechny soubory na webový server
